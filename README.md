## 메인컨셉 :

- 방어 (다가오는 바이러스 봇들을 파괴하여 메인 코어를 지켜야 한다)

### 서브 컨셉 1 :

- 로봇 (바이러스 봇들의 종류가 다양하다)

### 서브 컨셉 2 :

- 도구 (여러 무기와 바이러스를 없애는 도구들이 있다)

### 서브 컨셉 3 :

- 스테이지 (난이도별 스테이지가 있다)

### 서브 컨셉 4 :

- 가상현실 (모든 건 가상현실 속에 들어가 방어를 한다)

### 서브 컨셉 5 :

- 웨이브 (여러 차례로 점점 강력해지는 봇들이 다가온다)
